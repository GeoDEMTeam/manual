详细描述

Body is the class of objects, including particles and walls, in the simulation A Body instance have 4 main extensible data components

  1. Shape: shape
  2. Material: material
  3. State: physical state such as velocity, position, angular velocity, acceleration, etc.
  4. Bound: bound, generally used for pass 1 detection

Developers can extend the type of Body by implement subclass inherited from the 4 component classes
